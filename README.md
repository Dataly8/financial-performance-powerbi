# Análisis de rentabilidad y optimización de pricing

## 📊 Objetivo
Analizar la rentabilidad de productos y clientes para apoyar la toma de decisiones de negocio.

## 🔧 Proceso
- Modelado de datos en Power BI
- Creación de métricas mediante DAX
- Análisis de margen, descuentos y devoluciones

## 📈 Resultados
- Identificación de productos con bajo margen
- Detección de impacto de descuentos en rentabilidad
- Segmentación de productos y clientes

## 💡 Conclusiones
El análisis permitió identificar oportunidades de mejora en pricing y optimización de la rentabilidad.

## 🛠️ Herramientas
- Power BI
- DAX
- SQL (para análisis de datos)
